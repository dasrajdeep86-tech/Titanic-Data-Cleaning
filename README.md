# 🚢 Titanic Data Cleaning & Preprocessing

## 📌 Project Overview

This project was completed as part of the **Synent Technologies Data Science Internship Program**.

The objective of this project is to clean and preprocess the Titanic dataset to make it suitable for Exploratory Data Analysis (EDA) and Machine Learning.

---

## 🎯 Objectives

- Handle missing values
- Remove duplicate records
- Convert appropriate data types
- Rename column names
- Save the cleaned dataset

---

## 📂 Dataset

- Dataset: Titanic Dataset
- Source: Kaggle

---

## 🛠️ Tools & Libraries

- Python
- Pandas
- NumPy
- Google Colab

---

## 🔄 Workflow

1. Import Libraries
2. Load Dataset
3. Dataset Overview
4. Handle Missing Values
5. Remove Duplicate Records
6. Convert Data Types
7. Rename Columns
8. Save Clean Dataset

---

## 📊 Output

The cleaned dataset was saved as **Titanic_Cleaned.csv**.

The dataset is now ready for Exploratory Data Analysis (EDA) and Machine Learning.
